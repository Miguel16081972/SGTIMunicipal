Informe de Avances del Proyecto

Asunto: Resumen de funcionalidades y módulos implementados hasta la fecha.
Proyecto: Sistema de Gestión Territorial e Inteligencia Municipal (SGTI).

1. Aplicativos Móviles Desarrollados

📱 App Operativa (Campo)

Operativo
  • Detección automática de ubicación vía GPS.
  • Conversión de coordenadas a direcciones reales (calles).
  • Sincronización instantánea de reportes con la central municipal.
  • Interfaz simplificada para uso en exteriores.

📊 App de Gestión (Gerencia)

Operativo
  • Acceso remoto al tablero de control para autoridades.
  • Visualización de estadísticas y cumplimiento de metas.
  • Mapa de incidencias en tiempo real desde el celular.
  • Resúmenes ejecutivos para toma de decisiones.

2. Inteligencia de Datos (WhatsApp)

Se ha implementado un motor inteligente que procesa la información recibida por los canales vecinales:

  • Clasificación Automática: Los mensajes se asignan solos a gerencias como Seguridad, Limpieza o Rentas.
  • Detección de Urgencias: El sistema identifica palabras clave para priorizar casos críticos (Prioridad Alta).
  • Mapeo Automático: Extracción de direcciones del texto para ubicarlas sin intervención humana.

3. Mapa Territorial y Dashboard Central

  • Visualización Unificada: Un solo panel que integra reportes de vecinos (WhatsApp) y del personal municipal (App).
  • Gestión de Casos: Herramientas para actualizar el estado de cada incidente (Atendido, En Proceso, Nuevo).
  • Análisis Geográfico: Mapa interactivo con filtros por sectores y tipo de incidencia.

4. Gestión de Información y Reportes

  • Exportación de Datos: Módulo para descargar toda la gestión en formato Excel para estadísticas oficiales.
  • Base de Datos Segura: Almacenamiento organizado de todo el histórico de incidentes del distrito.
  • Vinculación de Canales: Panel para asociar grupos de WhatsApp específicos a las áreas responsables.
